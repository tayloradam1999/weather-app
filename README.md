# weather-app

A React-Typescript application that allows users to enter any city name and get the current weather.
  
Weather data is provided by [openweathermap.org](https://openweathermap.org/).
  
Weather data:
- Skies
- Temperature (K/C/F)
- Humidity
- Pressure
- Wind speed (m/s)
